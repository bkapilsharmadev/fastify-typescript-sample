## **Project Setup**

### **1. Clone the Repository**
```bash
git clone https://github.com/bkapilsharmadev/fastify-typescript-sample.git
cd fastify-typescript-sample
```

### **2. InstallDependencies**
```bash
npm install
```

### **3. Run the project**
- For Development
```bash
npm run dev
```
- For Production
```bash
npm run build && npm start
```
